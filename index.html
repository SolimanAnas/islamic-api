API Endpoints

Everything You Need

Seven domains covering all essential Islamic data. Search, paginate, and query with ease.

📖

Quran

Full Uthmani script text for all 114 surahs and 6,236 ayahs with search, juz, and page endpoints.

GET/v1/quran/surahs
GET/v1/quran/surah/{id}
GET/v1/quran/search?q={text}
📚

Hadith

13 authenticated collections from Bukhari to Nawawi's 40. Search across all or filter by collection.

GET/v1/hadith/collections
GET/v1/hadith/{collection}/{id}
GET/v1/hadith/search?q={text}
🏰

Adhkar

Hisn Muslim's 132 chapters plus morning, evening, sleeping, and post-salah remembrances.

GET/v1/azkar/hisn/chapters
GET/v1/azkar/morning
GET/v1/azkar/search?q={text}
📝

Tafsir

Four classical tafsir books: Ibn Kathir, al-Qurtubi, al-Baghawi, and as-Sa'di.

GET/v1/tafsir/books
GET/v1/tafsir/{book}/{surah}/{ayah}
GET/v1/tafsir/{book}/search?q={text}
🕌

Prayer Times

Accurate prayer times for 287+ cities worldwide, powered by the Aladhan API.

GET/v1/prayer/times?city={city}
GET/v1/prayer/times?lat={lat}&lon={lon}
GET/v1/prayer/cities
📻

Audio

150+ Quran radio stations and 40+ reciters with streaming URLs.

GET/v1/audio/stations
GET/v1/audio/reciters
Quick Start

Start Building in Seconds

No API key required. Just make a request and get your data.

terminal
# Get all 114 Quran surahs curl https://islamic-api.fly.dev/v1/quran/surahs # Search Hadith across all collections curl "https://islamic-api.fly.dev/v1/hadith/search?q=prayer" # Get Hisn Muslim chapters curl https://islamic-api.fly.dev/v1/azkar/hisn/chapters # Get tafsir for Ayat al-Kursi curl https://islamic-api.fly.dev/v1/tafsir/ibn-kathir/2/255 # Get prayer times for Cairo curl "https://islamic-api.fly.dev/v1/prayer/times?city=Cairo"
